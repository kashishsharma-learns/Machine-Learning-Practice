This repository contains a simple implementation of a **Decision Tree Classifier** using **scikit-learn** on the classic **Iris dataset**.  
The project demonstrates training the model, printing decision rules, and predicting the class of a new sample.

---

## Overview

- Dataset: Iris (built-in from `sklearn`)
- Algorithm: Decision Tree Classifier
- Splitting Criterion: Entropy (Information Gain)
- Output:
  - Human-readable decision rules
  - Predicted flower class for a sample input

---

- Python 3.7+
- scikit-learn
