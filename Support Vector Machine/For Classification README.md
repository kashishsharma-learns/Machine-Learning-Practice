This project demonstrates a **Support Vector Machine (SVM)** classifier using a **linear kernel** on the **Iris dataset**.  
The model is trained on two features (sepal length and sepal width) and visualized using a scatter plot.

---

## Dataset

- **Source:** `sklearn.datasets.load_iris`
- **Samples:** 150
- **Features Used:**
  - Sepal length
  - Sepal width
- **Classes:**
  - Setosa
  - Versicolor
  - Virginica

---

## Model Details

- **Algorithm:** Support Vector Machine (SVC)
- **Kernel:** Linear
- **Train/Test Split:** 70% / 30%
- **Random State:** 42
- **Evaluation Metric:** Accuracy
